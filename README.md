# K8s for SEs Workshop

Welcome, This is an workshop for RKE2/Rancher/Longhorn/Portworx/Gitea based on an opensource framework [HobbyFarm](https://github.com/hobbyfarm/hobbyfarm).

## User Interface

**https://hobbyfarm.rfed.io**

default password **Pa22word**

Each Student will get:

* 3 VMS :
  * **server** ( Rocky 9.4 ) - Control Plane/etcd/Worker
  * **worker1** ( Rocky 9.4 ) - Worker
  * **worker2** ( Rocky 9.4 ) - Worker

We are Deploying:

* **RKE2** ( STIG'd ) - Kubernetes
* **Rancher** - Multi Cluster Manager
* **Longhorn** - Stateful Storage
* **Portworx** - PX-CSI for connecting to an array
* **Gitea** - Version Control - extra credit

We are building:
* 2 clusters:
  * **local** 3 nodes/vms

Check out [Content - Step 1](https://github.com/clemenko/hobbyfarm/blob/main/workshop/content/step-1.md) of the content.

Hope you enjoy.
